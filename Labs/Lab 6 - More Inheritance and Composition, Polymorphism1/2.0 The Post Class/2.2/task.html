~2.0 The Post Class~

In this section you will be building the Post class that will form the basis of this lab's Lab Problem.

~Task 2.2~

It is a good idea to include accessor methods for all of Post's fields.

Post has some important methods:

  • react():
    • This method adds a specified reaction (one of the emojis) to reactions.
    • This method does not return anything.
  • addComment():
    • This method adds a new comment to comments.
    • This method does not return anything. A CommentPost is passed as a parameter.
  • display():
    • This method displays the Post and all of its relevant information textually.
    • The format of a Post will look like this:

      Chris Mascarolls on (Sat Dec 24 20:47:32 EST 2020):
      'Deck the halls.'
      =)(32) =|(13) =((1) =O(0)
      Comments:
      -> 'What are your plans?!' - Wanda Nauwittol (Sat Dec 24 20:47:33 EST 2020)
      -> 'Yay!' - Jude Alkingtomie (Sat Dec 24 20:48:32 EST 2020)
      -> 'Nope!' - Grinch Grinch (Sat Dec 24 24:48:32 EST 2020)

      The String format is already created for you in this section's Post class. You just need to insert the correct information inside String.format(). Note: for validation purposes, just put in "DATE"; the date will vary depending on when you complete this lab.
    • You can get the time in the above format by consulting the documentation for the java.util.Calendar class: Oracle Docs: Calendar.
    • Remember to bring over Comment.java from Task 2.1.
